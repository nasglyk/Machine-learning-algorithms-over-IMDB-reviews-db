Η εργασία μας αφορά την υλοποίηση δύο αλγορίθμων μηχανικής μάθησης. Συγκεκριμένα, υλοποιήσαμε τους αλγορίθμους ID3 και naive bayes που ταξινομούν αξιολογήσεις χρηστών ανάμεσα σε θετικές και αρνητικές γνωρίζοντας ένα πλήθος απο τις πιο χρησιμοποιημένες λέξεις.

Οι παράμετροι ζητούνται απο το χρήστη κατά την εκτέλεση του προγράμματος και αφορούν το πλήθος των λέξεων που θα περιέχει το λεξιλόγιο μας καθώς και το πλήθος των λέξεων που θέλουμε να κάνουμε skip.
 
Για την εκτέλεση τους κατεβάσαμε το TensorFlow package για να χρησιμοποιήσουμε το keras.
